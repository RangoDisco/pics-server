## Pic-Nic back-end code

# Run the project

## 1. Clone the project

## 2. Add .env variables

- DB_HOST
- DB_USERNAME
- DB_PASSWORD
- DB_NAME
- DB_CONTAINER_PORT
- HOST_PORT
- JWT_SECRET

## 3. Run with docker

```bash
yarn docker
```

## 4. Open playground or use Insomnia/Postman

Open [http://localhost:4000/graphql](http://localhost:4000/graphql) with your browser to see the result.
